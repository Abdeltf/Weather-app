<H1>Weather App Germany – PHP + OpenWeather API </H1> 

A simple but functional weather application built with PHP, cURL, HTML/CSS, and the OpenWeather API.
The user selects a German city, the backend requests live weather data, and the result is displayed directly on the page.

This project demonstrates basic API integration, form handling, server-side processing, JSON decoding, and clean UI structure.

<H1>Features</H1> 

<ul>
<li> Select a German city (Berlin, Frankfurt, München, Hamburg, Köln) </li>

<li> Backend request to OpenWeather API using PHP + cURL </li>

<li> Displays temperature and weather description </li>

<li> Validates user input </li>

<li> Simple, responsive UI </li>

</ul>

<H1> API Requirements </H1> 

To run the project, you need a free API key from OpenWeatherMap:

<ul>
<li> Create an account:
https://openweathermap.org/api </li>

<li> Generate an API key </li>

<li> Insert it into the PHP code </li>

</ul>
